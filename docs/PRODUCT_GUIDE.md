# ThingsPath 产品说明

> v1.1.0 · 2026-06-09

**ThingsPath** 是一款智能物品管理 App，帮你记录、追踪和分析所有个人物品——从电子产品到家具家电，让你清楚知道「我有什么、值多少钱、用了多久」。

---

## 界面概览

ThingsPath 采用极简设计，核心流程只有三步：**添加 → 查看 → 分析**。

主界面由以下区域组成：

| 区域 | 功能 |
|------|------|
| 统计卡片 | 显示物品总数和总价值，点击进入统计页 |
| 搜索栏 | 按名称、备注、位置搜索 |
| 标签筛选 | 横向滚动的分类标签，点击快速过滤 |
| 物品列表 | 每张卡片展示缩略图、名称、价格、每日成本、使用天数 |
| 添加按钮 | 右下角"+"按钮，打开 AI 智能添加 |

---

## 一、AI 智能添加 ✨

> 核心亮点功能——不用手动填表，说一句话就能录入物品。

### 怎么用

1. 点击主界面右下角的 **"+"** 按钮
2. 在弹出的对话框中，用自然语言描述你的物品，比如：
   - 「昨天在苹果店花 14999 买了台 MacBook Pro」
   - 「京东 299 入了一台空气净化器，放在卧室」
   - 「上周花 200 块买了双跑鞋」
3. 点击 **「分析」** 按钮
4. AI 自动提取物品名称、价格、购买日期、位置、分类标签等信息
5. 物品自动保存到列表中

### 小贴士

- 支持一次描述多个物品（AI 会批量提取）
- 支持相对时间：「昨天」「上周」「前天」等
- AI 会根据物品描述自动分到 15 个大类（如数码产品、家用电器、服装等）
- 需要在设置中配置 SiliconFlow API Key 才能使用

---

## 二、物品列表与搜索

### 搜索

- 在搜索栏输入关键词，实时过滤物品
- 支持按名称、备注、位置搜索

### 标签筛选

- 搜索栏下方显示所有分类标签
- 点击标签，只显示该分类下的物品
- 再次点击取消筛选

### 排序

- 点击筛选图标，打开排序菜单
- 可按以下维度排序：
  - 更新时间（默认）
  - 购买日期
  - 物品名称
  - 使用天数
  - 添加时间
- 点击同一排序项可切换升序/降序

### 下拉刷新

- 下拉页面即可从云端同步最新数据

### 批量操作

- **长按**任一物品卡片进入选择模式
- 顶部显示「已选择 N 项」
- 支持「全选」和「批量删除」
- 点击关闭按钮退出选择模式

---

## 三、物品详情

点击任意物品卡片进入详情页。

### 查看模式

展示物品的完整信息：

| 信息项 | 说明 |
|--------|------|
| 图片 | 横向滑动浏览多张图片，点击可全屏查看 |
| 名称 | 物品名称 |
| 位置 | 购买或存放位置 |
| 购买日期 | 格式：2024年6月1日 |
| 价格 | 购买价格 |
| 使用天数 | 从购买日至今的天数 |
| 每日成本 | 价格 ÷ 使用天数，了解「每天花多少钱」 |
| 备注 | 自由文字记录 |
| 标签 | 分类标签 |
| 添加日期 | 录入系统的日期 |

### 编辑模式

点击右上角编辑图标进入编辑模式，可修改所有信息：

- **图片管理**：支持从相册选择或拍照添加图片，可添加多张
- **购买日期与使用天数联动**：修改日期自动重算使用天数，反之亦然
- **标签管理**：输入文字后点"+"添加，点"×"移除
- 名称为必填项，保存时会校验

### 删除

- 在查看模式下点击删除图标
- 弹出确认对话框，确认后删除

---

## 四、统计分析

点击主界面的统计卡片进入。

### 提供三种可视化图表

| 图表 | 内容 |
|------|------|
| 热门标签 | 环形图展示前 5 个分类的物品数量分布 |
| 价格区间 | 环形图按「<1000」「1000-3000」「>3000」三档统计 |
| 位置分布 | 环形图展示前 6 个位置的物品数量分布 |

顶部还显示**物品总数**和**总价值**的大数字汇总。

---

## 五、设置

点击主界面右上角齿轮图标进入。

### AI 配置

- 填入 SiliconFlow API Key
- Key 仅存储在本设备，不会上传到任何服务器

### 数据恢复

- 点击「从云端恢复」，将云端数据库的数据同步到本地
- 适用于换机或重装后恢复数据

### 数据导入/导出

| 操作 | 说明 |
|------|------|
| 导出 | 将所有物品导出为 `thingspath_backup.json` 文件 |
| 导入 | 选择 JSON 备份文件，将物品导入（注意：重复导入会产生重复记录） |

---

## 隐私说明

- 所有物品数据默认存储在手机本地（Room 数据库）
- 图片存储在应用私有目录，不会出现在相册中
- 云端同步使用你自己的 Cloudflare D1 数据库和 R2 存储
- API Key 仅保存在本地设备，不会外传
- AI 分析通过 SiliconFlow API 完成，分析文本会发送到 AI 服务

## 技术特性

| 特性 | 技术 |
|------|------|
| 界面框架 | Jetpack Compose + Material 3 |
| 本地数据库 | Room (SQLite) |
| 云端数据库 | Cloudflare D1 |
| 图片云存储 | Cloudflare R2 |
| AI 服务 | SiliconFlow（兼容 OpenAI 格式） |
| 依赖注入 | Hilt / Dagger |
| 最低系统 | Android 7.0 (API 24) |
| 暗色模式 | 完整支持，跟随系统设置 |
